# Student Grade Tracker (GUI Based)

## Description
Student Grade Tracker is a Java Swing-based GUI application that allows users to manage student grades efficiently. The application enables users to enter the number of students, add student details and marks, and generate a summary report containing average, highest, and lowest scores.

## Features
- GUI-based interface using Java Swing
- Enter the number of students
- Add student names and marks
- Store student data using ArrayList
- Calculate average marks
- Find highest marks
- Find lowest marks
- Display a summary report
- Clear all entered data

## Technologies Used
- Java
- Java Swing
- ArrayList
- Object-Oriented Programming (OOP)

## How to Run

### Compile
```bash
javac StudentGradeTrackerGUI.java
```

### Run
```bash
java StudentGradeTrackerGUI
```

## Sample Workflow
1. Enter the number of students.
2. Click **Set Students**.
3. Enter student name and marks.
4. Click **Add Student**.
5. Repeat until all students are added.
6. Click **Show Report** to view:
   - Total Students
   - Average Marks
   - Highest Marks
   - Lowest Marks
7. Use **Clear** to reset the application.
